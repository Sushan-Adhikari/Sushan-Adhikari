### What I do

- **Research → production.** Low-resource NLP, LLM fine-tuning + RAG, and CV inference — published first, then deployed.
- **Computer vision at scale.** Optimized 5+ models on NVIDIA Triton and benchmarked DeepStream + GStreamer for real-time multi-camera pipelines (Computer Vision Intern → Junior MLOps Developer at KyraWorks).
- **MLOps.** CI/CD for model deployment with Docker, AWS, EC2, and Terraform.
- **Work that ships.** Integrated anti-spoofing liveness checks into Nepal's national e-pension platform (300K+ users), and built an ML job-recommendation pipeline across 50+ categories with 150+ global volunteers at Mercuri.world.
